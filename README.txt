ARCHIVE D'IMAGES GÉNÉRÉE

Contenu :
- images/ : toutes les images PNG
- index.html : galerie locale ou hébergeable
- links.txt : liens relatifs ou liens absolus si une adresse de site a été renseignée

Pour héberger les images, décompresse cette archive puis envoie le dossier images, et éventuellement index.html, sur le même hébergement.
